Método de Resolución para Homogéneas

El método estándar para resolver ecuaciones homogéneas involucra la sustitución y = vx, que convierte la ecuación en una forma separable.

Pasos del método:
1. Verificar que M y N sean homogéneas del mismo grado
2. Hacer la sustitución y = vx, dy = vdx + xdv
3. Sustituir en la ecuación original
4. Separar variables e integrar

🧮 Ejercicio resuelto:

Ejemplo:
Resolver: (x² - y²)dx + 2xy dy = 0

Paso 1: Verificar homogeneidad
M(x,y) = x² - y², N(x,y) = 2xy (ambas de grado 2)

Paso 2: Sustitución y = vx, dy = vdx + xdv
(x² - v²x²)dx + 2x(vx)(vdx + xdv) = 0
(1 - v²)dx + 2v(vdx + xdv) = 0

Paso 3: Simplificar
(1 + v²)dx + 2vx dv = 0
dx/x = -2v dv/(1 + v²)
Solución: x(1 + v²) = C → x + y²/x = C ✅
