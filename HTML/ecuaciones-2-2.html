Método del Factor Integrante

El factor integrante es la herramienta principal para resolver ecuaciones lineales. Se calcula como μ(x) = e^(∫P(x)dx) y permite transformar la ecuación en una forma integrable.

Fórmula general:
y = (1/μ(x))[∫μ(x)Q(x)dx + C]
donde μ(x) = e^(∫P(x)dx)

🧮 Ejercicio resuelto:

Ejemplo:
Resolver: y' + (1/x)y = x²

Paso 1: Identificar P(x) = 1/x, Q(x) = x²

Paso 2: Factor integrante
μ(x) = e^(∫(1/x)dx) = e^(ln|x|) = |x| = x (para x > 0)

Paso 3: Aplicar fórmula
y = (1/x)[∫x·x²dx + C] = (1/x)[∫x³dx + C]
y = (1/x)[x⁴/4 + C] = x³/4 + C/x ✅
