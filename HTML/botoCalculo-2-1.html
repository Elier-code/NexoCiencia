¿Qué es la Integración por Partes?

La técnica de integración por partes es una herramienta útil para resolver integrales que son producto de dos funciones. Está basada en la regla del producto para derivadas:

d(uv) = u·dv + v·du

De ahí se obtiene la fórmula de integración por partes:

∫u·dv = u·v - ∫v·du

¿Cuándo se usa?

Se utiliza cuando tenemos el producto de una función fácil de derivar (u) y otra fácil de integrar (dv).

Regla LIATE (sugerencia para elegir u):

  • Logaritmos: ln(x), log(x)
  • Inversas: arctan(x), arcsin(x)
  • Algebraicas: x, x², etc.
  • Trigonométricas: sin(x), cos(x), etc.
  • Exponenciales: ex, ax

Ejemplo resuelto

Ejemplo: Resolver ∫x·ex dx

  • u = x → du = dx
  • dv = ex dx → v = ex

Aplicamos la fórmula:
∫x·ex dx = u·v - ∫v·du
∫x·ex dx = x·ex - ∫ex dx
Resultado: x·ex - ex + C

Consejos útiles

  • Si después de aplicar una vez la técnica, la integral aún no es elemental, puedes repetir el método.
  • Es común aplicar esta técnica cuando tienes combinaciones como: x·ln(x), x·cos(x) o ln(x).
