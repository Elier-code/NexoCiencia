Introducción a las Ecuaciones de Bernoulli

¿Qué son las Ecuaciones de Bernoulli?

Las ecuaciones de Bernoulli son un tipo especial de ecuación diferencial no lineal de primer orden. Tienen la forma general:

y' + P(x)y = Q(x)y^n

donde n ≠ 0, 1. Estas ecuaciones son importantes porque:

  • Son no lineales cuando n ≠ 1
  • Se pueden transformar en ecuaciones lineales mediante sustitución
  • Tienen aplicaciones en dinámica de fluidos y crecimiento poblacional
  • Son fundamentales en muchos modelos matemáticos

Conceptos Clave

  • Identificación: Reconocer cuando una ecuación es de Bernoulli
  • Sustitución: Usar v = y^(1-n) para transformar la ecuación
  • Resolución: Aplicar métodos de ecuaciones lineales
  • Aplicaciones: Modelar fenómenos reales

Ejercicio Resuelto 1: Identificación y Transformación

Resolver la ecuación: y' + y = xy²

Solución:

Paso 1: Identificar la forma de Bernoulli

  • P(x) = 1
  • Q(x) = x
  • n = 2

Paso 2: Hacer la sustitución v = y^(1-n) = y^(-1)

Derivando: v' = -y^(-2)y' = -y^(-2)(xy² - y) = -x + y^(-1) = -x + v

Paso 3: Resolver la ecuación lineal resultante

v' - v = -x (ecuación lineal que se puede resolver con factor integrante)

Ejercicio Resuelto 2: Aplicación del Factor Integrante

Resolver: y' - 2y = 3y²

Solución:

Paso 1: Identificar parámetros

  • P(x) = -2
  • Q(x) = 3
  • n = 2

Paso 2: Sustitución v = y^(-1)

v' = -y^(-2)y' = -y^(-2)(2y + 3y²) = -2y^(-1) - 3 = -2v - 3

Paso 3: Resolver v' + 2v = -3

Factor integrante: e^(2x)

Solución: v = -3/2 + Ce^(-2x)
