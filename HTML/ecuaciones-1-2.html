Ejercicios de Ecuaciones de Bernoulli

Ejercicios Guiados

Ahora que conoces los conceptos básicos, practiquemos con ejercicios paso a paso para dominar la resolución de ecuaciones de Bernoulli.

Ejercicio 1: Identificación y Resolución Básica

Resolver: y' - 2y = 3y²

Solución paso a paso:

Paso 1: Identificar parámetros

  • P(x) = -2
  • Q(x) = 3
  • n = 2

Paso 2: Sustitución v = y^(-1)

v' = -y^(-2)y' = -y^(-2)(2y + 3y²) = -2y^(-1) - 3 = -2v - 3

Paso 3: Resolver v' + 2v = -3

Factor integrante: e^(2x)

Solución: v = -3/2 + Ce^(-2x)

Ejercicio 2: Aplicación del Factor Integrante

Resolver: y' + (1/x)y = x²y³

Solución paso a paso:

Paso 1: Identificar parámetros

  • P(x) = 1/x
  • Q(x) = x²
  • n = 3

Paso 2: Sustitución v = y^(-2)

v' = -2y^(-3)y' = -2y^(-3)(-y/x + x²y³) = 2y^(-2)/x - 2x²

Paso 3: Resolver v' - 2v/x = -2x²

Factor integrante: x^(-2)

Solución: v = -x² + Cx²

Ejercicio 3: Con Condición Inicial

Resolver: y' + 2xy = 6x√y con y(0) = 4

Solución paso a paso:

Paso 1: Reescribir como Bernoulli

y' + 2xy = 6xy^(1/2), donde n = 1/2

Paso 2: Sustitución v = y^(1/2)

v' = (1/2)y^(-1/2)y' = (1/2)y^(-1/2)(-2xy + 6xy^(1/2))

v' = -xv + 3x

Paso 3: Resolver v' + xv = 3x

Factor integrante: e^(x²/2)

Solución: v = 3 + Ce^(-x²/2)

Con y(0) = 4: C = -1, y = (3 - e^(-x²/2))²
