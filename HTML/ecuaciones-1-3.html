Ejercicios Prácticos de Ecuaciones de Bernoulli

Ejercicios Prácticos Avanzados

Ahora aplicaremos los conceptos aprendidos en problemas más complejos y situaciones reales. Estos ejercicios te ayudarán a consolidar tu comprensión y prepararte para el examen.

Tipos de ejercicios prácticos:

  • Problemas con condiciones iniciales complejas
  • Aplicaciones en dinámica de fluidos
  • Modelos de crecimiento poblacional
  • Circuitos eléctricos no lineales

Ejercicio Práctico 1: Aplicación en Dinámica de Fluidos

Resolver: y' + (1/x)y = x²y³ con y(1) = 2

Solución paso a paso:

Paso 1: Identificar parámetros

  • P(x) = 1/x
  • Q(x) = x²
  • n = 3

Paso 2: Sustitución v = y^(-2)

v' = -2y^(-3)y' = -2y^(-3)(-y/x + x²y³) = 2y^(-2)/x - 2x²

Paso 3: Resolver v' - 2v/x = -2x²

Factor integrante: x^(-2)

Solución: v = -x² + Cx²

Con y(1) = 2: C = 5, y = 1/√(5x² - x²) = 1/√(4x²) = 1/(2x)

Ejercicio Práctico 2: Modelo de Crecimiento Poblacional

Resolver: y' - 3y = 2y^(1/2) con y(0) = 1

Solución paso a paso:

Paso 1: Reescribir como Bernoulli

y' - 3y = 2y^(1/2), donde n = 1/2

Paso 2: Sustitución v = y^(1/2)

v' = (1/2)y^(-1/2)y' = (1/2)y^(-1/2)(3y + 2y^(1/2))

v' = (3/2)y^(1/2) + 1 = (3/2)v + 1

Paso 3: Resolver v' - (3/2)v = 1

Factor integrante: e^(-3x/2)

Solución: v = -2/3 + Ce^(3x/2)

Con y(0) = 1: C = 5/3, y = (-2/3 + (5/3)e^(3x/2))²

Ejercicio Práctico 3: Circuito Eléctrico No Lineal

Resolver: y' + 2xy = 6x√y con y(0) = 4

Solución paso a paso:

Paso 1: Reescribir como Bernoulli

y' + 2xy = 6xy^(1/2), donde n = 1/2

Paso 2: Sustitución v = y^(1/2)

v' = (1/2)y^(-1/2)y' = (1/2)y^(-1/2)(-2xy + 6xy^(1/2))

v' = -xv + 3x

Paso 3: Resolver v' + xv = 3x

Factor integrante: e^(x²/2)

Solución: v = 3 + Ce^(-x²/2)

Con y(0) = 4: C = -1, y = (3 - e^(-x²/2))²
