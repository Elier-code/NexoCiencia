Casos Especiales y Verificación

Es importante reconocer casos especiales en ecuaciones homogéneas y verificar las soluciones obtenidas para asegurar la corrección del proceso.

Casos especiales:
• Cuando M(x,y) = 0 o N(x,y) = 0
• Ecuaciones que no son homogéneas pero se pueden transformar
• Verificación: sustituir la solución en la ecuación original
• Condiciones de frontera y singularidades

🧮 Ejercicio resuelto:

Ejemplo:
Resolver: (x³ + y³)dx - 3xy² dy = 0

Paso 1: Verificar homogeneidad
M(x,y) = x³ + y³, N(x,y) = -3xy² (ambas de grado 3)

Paso 2: Sustitución y = vx
(x³ + v³x³)dx - 3x(vx)²(vdx + xdv) = 0
(1 + v³)dx - 3v²(vdx + xdv) = 0

Paso 3: Simplificar y separar
(1 - 2v³)dx - 3v²x dv = 0
dx/x = 3v² dv/(1 - 2v³)
Solución: x(1 - 2v³) = C → x - 2y³/x² = C ✅
